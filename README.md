# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game where users play against the computer. The game follows classic rules and provides instant results with score tracking.

## 🚀 Features

* Play against the computer
* Random computer choices
* Instant result display (Win/Lose/Draw)
* Score tracking system
* Simple and responsive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎮 How to Play

1. Choose Rock, Paper, or Scissors
2. The computer will make a random choice
3. The winner is decided based on rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock

## 📦 Installation

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
